Adding javascript to HTML Page

there are following ways:

  1. Embedding code
  2. External file
1.embedding code

To add the javascript code into html page, we can use the script tag of the html that wrap around javascript code inside the html program.

2.External jsfile code

To add the javascript code into html page, we can also create a seperate file to hold the code of javascript with the (.js) extension and later include it into our HTML document using the src attribute of the script tag. it becomes very useful if we want to use the same code
